Load CSS and Js

Include this code in the head.essenceUI uses the FontAwesome and jQuery.

        
<link rel="stylesheet" href="essence_ui.css" type="text/css" media="all" />
<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/font-awesome/4.6.2/css/font-awesome.min.css">
<script src="https://code.jquery.com/jquery-2.2.3.min.js"></script>
<script src="essence_ui.js"></script>

Style

Container and Grid

[.container] and [.container--fluid] are done responsive centering.
[.row] and [.row--0][.row--xs]...and more,summarizes the column.
column class is coding in BEM, but it is the same as the Bootstrap is in use.

          
<div class="container">
  <div class="row">
    <div class="col__md--4">md4</div>
    <div class="col__md--8">md8</div>
  </div>
  <div class="row row--0">
    <div class="col__sm--2">sm2</div>
    <div class="col__sm--6">sm6</div>
    <div class="col__sm--4">sm4</div>
  </div>
</div>
<div class="container--fluid">
  <div class="col__xs--6">xs6</div>
  <div class="col__xs--6">xs6</div>
</div>

Text align

[.text_align---xxx] class will align the text .

          
<p class="text__align--center">center</p>
<p class="text__align--left">left</p>
<p class="text__align--right">right</p>

Title

Class such as the settings for the title and title size.

          
<h2 class="text__title">text__title</h2>
<h2 class="text__title text__title--xs">text__title--xs</h2>
<h2 class="text__title text__title--sm">text__title--sm</h2>
<h2 class="text__title text__title--md">text__title--md</h2>
<h2 class="text__title text__title--lg">text__title--lg</h2>
<h2 class="text__title text__title--xl">text__title--xl</h2>

Card

This class will adapt the card design.

          
<div class="card card--md card--center">
  <div class="card__header card__header--md">header</div>
  <div class="card__block--md">contents</div>
</div>
<div class="card card--dark">
  <div class="card__block--md">contents</div>
</div>
